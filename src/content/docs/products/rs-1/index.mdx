# OpticWorks RS-1

RS-1 is OpticWorks' presence sensor family built on a single PCBA with selective population. It runs HardwareOS (ESP-IDF firmware) and integrates with Home Assistant via the ESPHome Native API, with optional cloud services for OTA, telemetry, and zone editor sync.

**Variants:** RS-1 Lite (LD2410) and RS-1 Pro (LD2410 + LD2450). **Add-ons:** PoE module and IAQ daughterboard.
