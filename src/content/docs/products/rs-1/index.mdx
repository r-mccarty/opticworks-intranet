# OpticWorks RS-1

The RS-1 is OpticWorks' flagship vision + radar sensor fusion device built on hardwareOS. Its core capabilities (fusion, 30Hz WorldState streaming, Kalman filtering, Hungarian association, RoomPlan calibration) are documented in the hardwareOS repo.
