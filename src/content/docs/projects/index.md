## Core Repositories

| Repo | Purpose | Primary Stack | Notes |
|------|---------|----------------|-------|
| `hardwareOS` | Embedded platform for RS-1 (sensor fusion + WebRTC) | Go, C, React | See `hardwareOS/README.md` |
| `presence-detection-engine` | ESPHome/ESP32 presence engine + HA tooling | C++, Python, YAML | See `presence-detection-engine/README.md` |
| `opticworks-store` | E-commerce platform (storefront + Medusa backend) | Next.js, Medusa, PostgreSQL | See `opticworks-store/README.md` |
| `opticworks-intranet` | Internal docs site | Astro/Starlight | This repo |
| `n8n-marketing-automation` | Marketing automation via N8N | Docker, Python | See `n8n-marketing-automation/README.md` |
| `agent-harness` | Coder workspace bootstrap | Terraform + Docker | See `agent-harness/AGENTS.md` |
